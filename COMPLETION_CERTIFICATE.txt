  1. 验证系统状态:
     python verify_cleanup.py

  2. 启动后端服务:
     cd backend
     python -m uvicorn app.main:app --port 8001

  3. 访问 admin 页面:
     http://localhost:8001/admin/

  4. Docker 部署:
     docker-compose up -d
