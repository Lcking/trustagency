# TrustAgency OpenClaw 运营策略文档

## 站点概述

**TrustAgency (鹰眼查融)** 是一个专注于杠杆交易和融资融券信息的金融资讯网站。

### 网站地址
- 主站: https://www.yycr.net
- 两融数据: https://www.yycr.net/margin/

---

## 站点架构

### 内容结构

```
TrustAgency
├── 📰 内容栏目 (Sections)
│   ├── FAQ (常见问题) - 不需要关联平台
│   ├── Wiki (百科) - 不需要关联平台
│   ├── Guide (指南) - 不需要关联平台
│   └── Review (平台评测) - 需要关联具体平台
│
├── 🏢 平台库 (Platforms)
│   └── 各类杠杆交易平台信息
│
├── 📊 两融数据 (Margin Data)
│   ├── 市场汇总数据 (/margin/)
│   ├── 排行榜 (/margin/ranking/{order}/)
│   └── 个股详情 (/margin/stock/{ts_code}/)
│
└── 📂 分类体系 (Categories)
    └── 每个栏目下有多个分类
```

## 可用 API 接口

### 1. 外部任务 API (API Key 认证)

**认证方式**: `X-API-Key` Header

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/external/ping` | GET | 连通性测试（无需认证） |
| `/api/external/health` | GET | 系统健康检查 |
| `/api/external/tasks/sync-margin` | POST | 同步两融数据 |
| `/api/external/tasks/margin-status` | GET | 查询两融数据状态 |

### 2. 内容管理 API (JWT 认证)

**认证方式**: 需要先登录获取 JWT Token

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/admin/login` | POST | 管理员登录，获取 Token |
| `/api/sections` | GET | 获取所有栏目 |
| `/api/categories` | GET | 获取所有分类 |
| `/api/platforms` | GET | 获取所有平台 |
| `/api/articles` | GET/POST | 文章列表/创建 |
| `/api/tasks/generate-articles` | POST | 提交 AI 文章生成任务 |

## 运营策略建议

### 📊 数据同步任务

**任务**: 两融数据每日同步

| 触发时间 | 说明 |
|----------|------|
| 每天 17:30 | 第一次同步（Tushare 通常 17:00 后发布） |
| 每天 18:30 | 备用同步（防止数据延迟） |

**OpenClaw 配置**:
```yaml
任务名称: 两融数据同步
触发器: Cron "30 17 * * *" 和 "30 18 * * *"
动作: POST https://www.yycr.net/api/external/tasks/sync-margin
Headers:
  X-API-Key: {{secrets.TRUSTAGENCY_API_KEY}}
```

## AI 文章生成指南

### 生成流程

1. **准备标题列表**: 根据内容策略准备文章标题
2. **选择栏目和分类**: 确定文章归属
3. **提交生成任务**: 调用 API 提交任务
4. **监控任务状态**: 等待生成完成
5. **审核发布**: 人工审核后发布

### API 调用示例

**步骤 1: 登录获取 Token**

```bash
curl -X POST "https://www.yycr.net/api/admin/login" \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "your_password"}'
```

**步骤 2: 提交生成任务**

```bash
curl -X POST "https://www.yycr.net/api/tasks/generate-articles" \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "titles": ["什么是融资融券？", "融资利率怎么算？"],
    "section_id": 3,
    "category_id": 1,
    "ai_config_id": 1,
    "auto_publish": false
  }'
```

## 监控与告警

### 告警条件

| 条件 | 严重程度 | 建议动作 |
|------|----------|----------|
| 两融数据 > 3 天未更新 | 高 | 检查同步任务，手动触发 |
| 健康检查失败 | 高 | 检查服务器状态 |
| API 响应 > 10 秒 | 中 | 检查服务器负载 |
| 文章生成失败率 > 50% | 中 | 检查 AI 配置 |

### 监控仪表盘指标

- 两融数据最新日期
- 今日同步记录数
- 文章总数 / 本周新增
- AI 任务成功率
- API 响应时间
