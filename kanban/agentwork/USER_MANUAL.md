## 快速开始

### 首次登录

1. **打开系统**
   - 访问 `https://yourdomain.com` 进入登录页面

2. **输入凭证**
   - 用户名: 由管理员提供
   - 密码: 由管理员提供

3. **登录**
   - 点击"登录"按钮
   - 系统将验证凭证并跳转到仪表板

4. **修改密码** (推荐)
   - 点击右上角的用户菜单
   - 选择"设置"
   - 点击"修改密码"
   - 输入新密码并确认

### 首次使用建议

1. 阅读本手册的"系统界面导航"部分
2. 浏览示例平台和文章
3. 创建一个测试平台
4. 创建一篇测试文章
5. 尝试使用 AI 生成功能

## 平台管理

### 查看平台列表

1. **进入平台管理**
   - 点击导航栏中的"平台管理"

2. **浏览平台列表**
   - 列表显示所有已创建的平台
   - 默认按排名升序排列

3. **使用搜索功能**
   - 在搜索框中输入平台名称或关键词
   - 按回车键执行搜索
   - 点击清除按钮可清空搜索

4. **应用过滤器**
   - **活跃状态**: 只显示活跃/非活跃的平台
   - **精选**: 只显示精选平台
   - **排序**: 按名称、排名、评分等排序

### 创建新平台

1. **点击"新增平台"按钮**
   - 位于平台列表页面的顶部右侧

2. **填写基本信息**
   ```
   平台名称 *
   - 输入平台的名称 (如 "Binance")
   - 长度: 1-100 字符
   
   描述 *
   - 详细描述平台的特点
   - 长度: 最多 500 字符
   ```

3. **填写平台数据**
   ```
   排名 *
   - 平台在行业中的排名
   - 必须为正整数
   
   评分 *
   - 平台的评分
   - 范围: 0-5 分
   
   最小杠杆 *
   - 平台提供的最小杠杆倍数
   
   最大杠杆 *
   - 平台提供的最大杠杆倍数
   - 必须 >= 最小杠杆
   
   手续费率 *
   - 平台的交易手续费
   - 范围: 0-1 (0 = 0%, 1 = 100%)
   ```

4. **填写链接和配置**
   ```
   Logo URL *
   - 平台 Logo 的 URL
   - 必须是有效的 HTTPS 链接
   
   官网 URL *
   - 平台官方网站
   - 必须是有效的 HTTPS 链接
   
   是否受监管 *
   - 选择是/否
   
   是否精选 (可选)
   - 勾选以在前台显示为精选平台
   ```

5. **保存**
   - 点击"保存"按钮
   - 系统将验证所有字段
   - 成功后将显示新创建的平台

### 编辑平台信息

1. **进入平台列表**
   - 点击导航栏中的"平台管理"

2. **找到要编辑的平台**
   - 使用搜索或过滤功能
   - 点击平台名称或"编辑"按钮

3. **修改需要更改的字段**
   - 只需修改想要更改的字段
   - 其他字段保持不变

4. **保存更改**
   - 点击"保存"按钮
   - 成功后显示确认消息

### 删除平台

1. **在平台列表中**
   - 找到要删除的平台

2. **点击"删除"按钮**
   - 系统会显示确认对话框

3. **确认删除**
   - 点击"确认"按钮
   - 平台将被删除

⚠️ **警告**: 删除平台会删除所有相关的文章！

## 文章管理

### 查看文章列表

1. **进入文章管理**
   - 点击导航栏中的"文章管理"

2. **查看文章列表**
   - 显示所有文章及其状态
   - 每行显示一篇文章的基本信息

3. **使用搜索和过滤**
   ```
   搜索框
   - 搜索文章标题、内容或摘要
   
   分类过滤
   - 按分类筛选文章
   
   平台过滤
   - 按所属平台筛选
   
   状态过滤
   - 已发布/草稿
   - 精选/普通
   ```

### 创建新文章

1. **点击"新增文章"按钮**
   - 位于文章列表页面的顶部右侧

2. **选择所属平台**
   - 从下拉列表选择平台
   - 这将决定文章在前台的分类

3. **填写文章标题**
   ```
   标题 *
   - 清晰、吸引人的标题
   - 长度: 1-200 字符
   - 示例: "Bitcoin 初学者指南"
   ```

4. **填写文章内容**
   ```
   内容 *
   - 文章的主要内容
   - 支持 Markdown 格式
   - 最小长度: 100 字符
   - 建议: 1000-3000 字符
   
   Markdown 提示:
   - # 标题 1
   - ## 标题 2
   - **粗体** 和 *斜体*
   - [链接](url)
   - ![图片](url)
   - - 列表项
   - > 引用
   ```

5. **填写摘要**
   ```
   摘要 *
   - 文章的简短描述
   - 显示在列表和搜索结果中
   - 长度: 最多 500 字符
   - 示例: "快速了解 Bitcoin 的要点"
   ```

6. **分类和标签**
   ```
   分类 *
   - 文章的分类
   - 长度: 1-50 字符
   - 示例: "教程", "新闻", "指南"
   
   标签 (可选)
   - 用于更细粒度的分类
   - 支持最多 10 个标签
   - 每个标签最多 30 字符
   ```

7. **SEO 信息** (可选)
   ```
   SEO 描述
   - 搜索引擎显示的描述
   - 长度: 最多 160 字符
   
   SEO 关键词
   - 用于搜索引擎优化
   - 长度: 最多 100 字符
   ```

8. **保存文章**
   - 点击"保存为草稿"以保存为草稿
   - 点击"发布"以直接发布

### 编辑文章

1. **在文章列表中**
   - 找到要编辑的文章
   - 点击文章标题或"编辑"按钮

2. **修改文章内容**
   - 修改任何需要更改的字段

3. **保存更改**
   - 点击"保存"按钮
   - 如果已发布，直接更新；如果是草稿，保持为草稿

### 发布文章

1. **选择要发布的文章**
   - 在文章列表中找到文章
   - 状态应显示"草稿"

2. **点击"发布"按钮**
   - 或在编辑页面点击"发布"

3. **确认发布**
   - 系统会显示确认对话框
   - 点击"确认"

4. **完成**
   - 文章现在对外部用户可见
   - 状态会变为"已发布"

### 删除文章

1. **在文章列表中**
   - 找到要删除的文章

2. **点击"删除"按钮**
   - 系统会显示确认对话框

3. **确认删除**
   - 点击"确认"按钮
   - 文章将被删除

## AI 内容生成

### 提交生成任务

1. **进入 AI 生成**
   - 点击导航栏中的"AI 生成"

2. **点击"新建任务"按钮**

3. **输入要生成的文章标题**
   ```
   - 每行一个标题
   - 支持多个标题，最多 100 个
   - 示例:
     Python 入门指南
     FastAPI 最佳实践
     Docker 容器化入门
   ```

4. **选择文章分类**
   ```
   分类
   - guide (指南)
   - news (新闻)
   - tutorial (教程)
   ```

5. **添加批次名称** (可选)
   ```
   - 用于区分不同的生成批次
   - 示例: "November 2025 Batch"
   ```

6. **提交任务**
   - 点击"提交"按钮
   - 系统将显示任务 ID
   - 任务被加入处理队列

### 监控任务进度

1. **在 AI 生成页面**
   - 任务列表显示所有已提交的任务

2. **查看任务详情**
   - 点击任务 ID 或任务名称
   - 显示详细的进度信息

3. **进度指示**
   ```
   - 进度条显示完成百分比
   - 显示已完成/总数
   - 显示失败数 (如有)
   - 显示预计剩余时间
   ```

4. **任务状态**
   ```
   等待中 - 任务等待处理
   处理中 - 正在生成文章
   已完成 - 所有文章已生成
   失败 - 某些或所有文章生成失败
   已取消 - 任务已被用户取消
   ```

### 查看生成结果

1. **任务完成后**
   - 状态会更新为"已完成"

2. **点击"查看结果"**
   - 显示生成的文章列表

3. **查看生成的文章**
   - 每篇文章显示标题、摘要、状态
   - 可以直接编辑或发布

4. **发布文章**
   - 点击文章的"发布"按钮
   - 或编辑后再发布

### 取消任务

1. **在任务列表中**
   - 找到要取消的任务
   - 状态必须是"等待中"或"处理中"

2. **点击"取消"按钮**
   - 系统会显示确认对话框

3. **确认取消**
   - 任务将被取消
   - 已生成的部分文章将被保留

## 用户和权限

### 管理员用户

系统支持多个管理员账户：

- **超级管理员 (Superadmin)**
  - 拥有所有权限
  - 可以管理其他管理员
  - 可以访问系统设置

- **普通管理员 (Admin)**
  - 可以管理平台、文章、任务
  - 无法管理其他管理员
  - 无法访问系统设置

### 修改个人信息

1. **点击右上角的用户菜单**
   - 显示当前用户名

2. **点击"设置"**
   - 进入个人设置页面

3. **修改信息**
   ```
   用户名
   - 显示为只读
   
   电子邮件
   - 可以更改
   - 必须是有效的邮件地址
   
   全名
   - 可以更改
   - 长度: 最多 100 字符
   ```

4. **保存更改**
   - 点击"保存"按钮

### 修改密码

1. **在设置页面**
   - 点击"修改密码"按钮

2. **输入新密码**
   ```
   当前密码 *
   - 输入当前密码以确认身份
   
   新密码 *
   - 至少 8 个字符
   - 建议包含大小写字母、数字、特殊字符
   
   确认密码 *
   - 再次输入新密码
   ```

3. **保存**
   - 点击"保存"按钮
   - 密码将被更新

## 常见操作

### 导出数据

1. **在平台列表**
   - 点击"导出"按钮
   - 选择格式 (CSV/Excel)
   - 文件将被下载

2. **在文章列表**
   - 点击"导出"按钮
   - 可以选择导出所有或筛选后的文章

### 批量操作

1. **选择多个项目**
   - 勾选每行左侧的复选框
   - 或点击列表顶部的"全选"

2. **批量更新状态**
   - 在工具栏中选择操作
   - 选择新状态
   - 点击"应用"

### 撤销操作

- 大多数操作可以通过删除然后重新创建来撤销
- 某些操作 (如删除) 不可撤销，请谨慎操作

### 查看活动日志

1. **点击"分析"菜单**

2. **点击"活动日志"**

3. **查看最近的活动**
   - 显示操作者、操作类型、时间等

## 常见问题 FAQ

### Q: 我忘记了密码怎么办？
**A**: 点击登录页面的"忘记密码"链接，按照邮件中的说明重置密码。

### Q: 可以同时编辑一篇文章吗？
**A**: 不建议。如果多个用户同时编辑，最后保存的版本将被使用。

### Q: 删除的文章可以恢复吗？
**A**: 不可以。删除是永久的。在删除前请确认。

### Q: AI 生成文章需要多长时间？
**A**: 通常每篇文章需要 1-3 分钟。取决于服务器负载。

### Q: 我无法登录系统怎么办？
**A**: 
1. 检查用户名和密码是否正确
2. 检查 Caps Lock 是否打开
3. 清除浏览器 Cookie 和缓存
4. 尝试使用不同的浏览器
5. 联系管理员

### Q: 如何导出所有文章？
**A**: 在文章列表页面点击"导出"按钮，选择格式后下载。

### Q: 可以修改已发布的文章吗？
**A**: 可以。编辑后点击"保存"，更改会立即生效。

### Q: 如何在文章中插入图片？
**A**: 
1. 在内容编辑框中使用 Markdown 语法
2. `![图片描述](图片URL)`
3. 图片 URL 必须是有效的 HTTPS 链接

### Q: 系统支持多语言吗？
**A**: 目前只支持中文。未来版本将添加多语言支持。

### Q: 如何查看文章的浏览统计？
**A**: 在"分析"菜单中查看详细的浏览统计。
