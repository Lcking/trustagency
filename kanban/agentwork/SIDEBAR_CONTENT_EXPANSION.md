### CSS 类说明

| 类 | 作用 |
|----|------|
| `card` | 卡片容器 |
| `mb-4` | 下方边距 (1.5rem) |
| `card-header` | 卡片头部 |
| `bg-light` | 浅灰色背景 |
| `card-title` | 标题样式 |
| `h6` | 标题大小 |
| `list-group` | 列表组 |
| `list-group-flush` | 移除边框 |
| `small` | 小字体 |
| `text-decoration-none` | 移除下划线 |

## 💡 设计考虑

### 为什么选择这个结构?

1. **优先级递减**: 相关资源 → 热门百科 → 热门指南
2. **视觉区分**: 使用 `bg-light` 区分新卡片
3. **一致性**: 所有文件采用相同结构
4. **可维护性**: 易于添加或修改链接
5. **用户体验**: 最重要的内容放在顶部
