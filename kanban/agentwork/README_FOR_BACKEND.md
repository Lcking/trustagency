```
    ┌───────────────────────────┼───────────────────────────┐

│                                             │  └──────────────────┘
│  ├─ /api/admin/login (认证)               │
│  ├─ /api/platforms/* (平台管理)           │
│  ├─ /api/articles/* (文章管理)            │
│  ├─ /api/admin/generate/* (AI 生成) ⭐   │
│  └─ /health (健康检查)                     │
└─────────┬───────────────────────────────────┘
          │
          ├─────────────────────────────┬──────────────┐
          ▼                             ▼              ▼
      ┌────────────┐         ┌──────────────────┐  ┌─────────┐
      │ SQLite/    │         │   Celery + Redis │  │OpenAI   │
      │PostgreSQL  │         │  (后台任务队列)  │  │API      │
      └────────────┘         └──────────────────┘  └─────────┘
        (数据库)                (异步处理)          (AI 模型)
```

**选型理由**:
- FastAPI: 异步性能最强，AI 生态最好
- Celery + Redis: 处理长时间 AI 生成任务
- SQLite/PostgreSQL: 简单可靠，易于备份
- OpenAI: 最成熟的商用 AI 模型

### 创建项目
```bash
cd trustagency
mkdir backend
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 运行开发环境
```bash
# 终端 1: Redis
redis-server

# 终端 2: FastAPI
python -m uvicorn app.main:app --reload

# 终端 3: Celery Worker
celery -A app.celery_app worker --loglevel=info
```
