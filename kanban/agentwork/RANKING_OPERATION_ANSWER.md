### 编辑平台排名

#### 场景：新增了 5 个平台，需要设置排名

**初始状态**:
```
平台列表:
#1 Alpha Leverage (排名: 1)
#2 Beta Margin (排名: 2)
#3 New Platform 1 (排名: 空)  ← 需要排名
#4 New Platform 2 (排名: 空)  ← 需要排名
#5 New Platform 3 (排名: 空)  ← 需要排名
#6 New Platform 4 (排名: 空)  ← 需要排名
#7 New Platform 5 (排名: 空)  ← 需要排名
```

**编辑步骤**:

```
1️⃣  点击 "🏢 交易平台" 菜单
     ↓
   看到平台列表

2️⃣  找到 "New Platform 1" 
     ↓
   点击这一行后面的 ✏️ 编辑按钮

3️⃣  弹出编辑对话框
     ├─ 平台名称: New Platform 1
     ├─ 评分: 3.5
     ├─ 排名: [  空的输入框  ] ← 点这里
     ├─ 佣金: 0.4%
     └─ ... 其他字段 ...

4️⃣  在排名字段输入: 3
     ↓
   排名: [ 3 ]

5️⃣  点 [保存] 按钮
     ↓
   ✅ 完成！返回列表

6️⃣  重复步骤 2-5 处理其他 4 个平台
   - New Platform 2 → 排名 4
   - New Platform 3 → 排名 5
   - New Platform 4 → 排名 6
   - New Platform 5 → 排名 7
```

**最终状态**:
```
平台列表:
#1 Alpha Leverage (排名: 1) ✓
#2 Beta Margin (排名: 2) ✓
#3 New Platform 1 (排名: 3) ✓
#4 New Platform 2 (排名: 4) ✓
#5 New Platform 3 (排名: 5) ✓
#6 New Platform 4 (排名: 6) ✓
#7 New Platform 5 (排名: 7) ✓

全部完成！
```
