## ⌨️ 键盘快捷键参考

### 键盘导航
```
Tab              → 下一个元素
Shift + Tab      → 上一个元素
Enter            → 激活链接/按钮
Space            → 激活按钮/复选框
Escape           → 关闭菜单/模态
```

### VoiceOver (macOS)
```
Cmd + F5         → 启用/禁用 VoiceOver
VO + ↑/↓         → 上/下一个元素
VO + 空格         → 激活元素
VO + `           → 打开转子菜单
H                → 下一个标题
L                → 下一个链接
B                → 下一个按钮
```
