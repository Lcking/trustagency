# 管理后台操作指南

## 快速开始

### 访问管理后台
1. 打开 `http://localhost:8001/admin/`
2. 用户名: `admin`
3. 密码: `admin123`
4. 点击"登录"

### 仪表板 (首页)
显示系统整体状态：
- **平台总数**: 已创建的交易平台数量
- **文章总数**: 所有文章数量
- **已发布**: 已发布文章数量  
- **进行中任务**: 正在处理的AI生成任务
- **总浏览量**: 所有文章的总浏览量
- **系统状态**: 后端、数据库、缓存、API连接状态

---

## 平台管理

### 查看平台列表
1. 点击左侧菜单 "平台管理"
2. 显示所有平台的列表，包含:
   - 名称
   - 网址
   - 状态 (活跃/禁用)
   - 操作按钮

### 创建新平台
1. 点击 "+ 新增平台" 按钮
2. 填写以下信息:
   - **平台名称** (必填): 如 "Binance"
   - **网址**: 平台官方URL
   - **评分**: 1-5星评分
   - **排名**: 平台排序位置
   - **最小杠杆**: 如 1.0
   - **最大杠杆**: 如 500.0
   - **描述**: 平台简介
   - **受监管**: 勾选如果平台受监管
   - **活跃状态**: 勾选保持平台活跃
3. 点击 "保存" 按钮
4. 成功后,新平台会出现在列表中

### 编辑平台
1. 在平台列表中点击平台行上的 "编辑" 按钮
2. 修改需要更新的字段
3. 点击 "保存"

### 删除平台
1. 在平台列表中点击 "删除" 按钮
2. 确认删除
3. 平台被移除

---

## 文章管理

### 查看文章列表
1. 点击左侧菜单 "文章管理"
2. 显示所有文章，包含:
   - 标题
   - 分类
   - 浏览量
   - 发布状态 (草稿/已发布)
   - 操作按钮

### 创建新文章
1. 点击 "+ 新增文章" 按钮  
2. 填写文章信息:
   - **文章标题** (必填): 文章主标题
   - **所属平台** (必填): 选择关联的平台
   - **分类** (必填): 如 "教程"、"新闻"、"评测"
   - **摘要**: 文章简介 (可选)
   - **内容** (必填): 文章正文
   - **立即发布**: 勾选则文章发布,不勾选为草稿
   - **精选文章**: 勾选标记为精选
3. 点击 "保存"
4. 文章会出现在列表中

### 编辑文章
1. 在文章列表中点击 "编辑" 按钮
2. 模态窗口会打开,所有字段预填充
3. 修改需要更新的内容
4. 点击 "保存" 提交

### 删除文章
1. 在文章列表中点击 "删除" 按钮
2. 确认删除
3. 文章被删除

### 搜索文章
1. 在 "搜索文章..." 输入框中输入关键词
2. 按Enter或自动搜索
3. 列表会筛选匹配的文章

### 刷新列表
点击 "刷新" 按钮重新加载最新数据

## AI 任务 (批量生成)

### 查看任务列表
1. 点击左侧菜单 "AI任务"
2. 显示所有生成任务的状态

### 创建批量生成任务
1. 在 "批量生成文章" 部分:
   - **选择平台**: 从下拉菜单选择目标平台
   - **输入分类**: 输入文章分类
   - **输入标题**: 一行一个标题,输入多个标题来批量生成
2. 点击 "🚀 提交生成任务"
3. 系统会开始处理,可在任务列表中查看进度

### 查看任务详情
1. 在任务列表中点击 "详情" 按钮
2. 显示任务进度、完成数量等信息

## 常见问题

### Q: 如何更改登录密码?
A: 目前需要在数据库中手动更改或通过API。

### Q: 文章保存后为什么没有立即显示?
A: 刷新页面或点击"刷新"按钮重新加载列表。

### Q: 能删除已关联文章的平台吗?
A: 需要先删除所有相关文章,才能删除平台。

### Q: 编辑后的文章什么时候生效?
A: 点击保存后会立即生效,前端会自动刷新列表。

## 数据验证规则

### 平台
- 名称: 必填,最大255字符
- 网址: 可选,最大500字符
- 评分: 0-5之间的数字
- 杠杆范围: 最小>0,最大>最小

### 文章
- 标题: 必填,最大255字符
- 分类: 必填,最大100字符
- 内容: 必填
- 摘要: 可选,用于列表预览
- 平台: 必填,必须关联到现有平台

## 技术细节

### 认证
- 使用JWT令牌
- 令牌有效期: 根据环境变量配置
- 自动登录会话: 浏览器本地存储

### API 端点
- `/api/platforms` - 平台管理
- `/api/articles` - 文章管理  
- `/api/tasks/generate-articles` - AI生成任务

### 数据库
- PostgreSQL (生产环境)
- SQLite (开发环境)
- SQLAlchemy ORM
