### 第四步：输入排名并保存

**修改步骤**:

```
1. 找到 "排名" 字段
2. 清空当前值 (如果有)
3. 输入新排名: 3

最终效果:
排名: 3 ✓

4. 点击 [保存] 按钮
```

**保存后**:

## 🔄 批量操作步骤 (所有 5 个新平台)

假设你想要这样的排名：

```
New Platform 1 → 排名 3
New Platform 2 → 排名 4
New Platform 3 → 排名 5
New Platform 4 → 排名 6
New Platform 5 → 排名 7
```

**逐个操作** (每个 30 秒):

```
1️⃣  New Platform 1
   ✏️ 编辑 → 输入排名 3 → 保存 (30 秒)

2️⃣  New Platform 2

总耗时: 5 × 30 秒 = 2.5 分钟
```

## 💡 更简便的办法 - 直接修改 SQLite

如果你有很多平台要排名，还可以直接修改数据库：

### 方法 1：使用 SQLite 图形工具

```
1. 下载 DB Browser for SQLite
   https://sqlitebrowser.org/

2. 打开 trustagency.db 文件
   /Users/ck/Desktop/Project/trustagency/backend/trustagency.db

3. 找到 platforms 表

4. 直接编辑 rank 列:
   ┌──────┬──────────────────┬─────────────┐
   │ ID   │ name             │ rank        │
   ├──────┼──────────────────┼─────────────┤
   │ 3    │ New Platform 1   │ 3 (输入)   │
   │ 4    │ New Platform 2   │ 4 (输入)   │
   │ 5    │ New Platform 3   │ 5 (输入)   │
   │ 6    │ New Platform 4   │ 6 (输入)   │
   │ 7    │ New Platform 5   │ 7 (输入)   │
   └──────┴──────────────────┴─────────────┘

5. 点击 [保存] 按钮

6. 关闭应用

完成！改动会立即生效。
```

```
├─ 官网 URL
├─ 最小杠杆倍数

```
