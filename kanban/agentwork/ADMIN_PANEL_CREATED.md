## 🔧 故障排除

### 如果无法访问
1. 检查后端是否运行
   ```bash
   docker-compose ps
   ```

2. 查看后端日志
   ```bash
   docker-compose logs backend
   ```

3. 检查端口 8001 是否开放
   ```bash
   lsof -i :8001
   ```

### 如果登录失败
1. 清除浏览器缓存
2. 尝试使用隐私窗口
3. 检查控制台错误 (F12)
4. 验证后端 API 是否正常

### 如果看不到数据
1. 首先需要添加数据
2. 通过 API 创建平台和文章
3. 然后在面板中查看
