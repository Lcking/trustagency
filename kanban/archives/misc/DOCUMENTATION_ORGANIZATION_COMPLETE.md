## 🛠️ 使用方法

### 查找文档

**方法 1：VS Code 文件浏览器**
```
1. 打开 VS Code
2. 导航到 kanban/archives/
3. 选择相应分类查看文件
```

**方法 2：命令行**
```bash
# 进入文档目录
cd kanban/archives/

# 列出所有分类
ls -la

# 查看某个分类的文件
ls -la deployments/

# 搜索文件
find . -name "*deployment*"
```

**方法 3：全局搜索**
```bash
# VS Code 快捷键：Ctrl+Shift+F
# 搜索内容：例如 "memory", "docker", "deployment"
```

### 添加新文档

1. 创建新文档
2. 根据内容确定分类
3. 放在相应的 `archives/{category}/` 目录
4. 提交到 Git

```bash
# 示例
touch kanban/archives/bug_fixes/BUG_XXX_TITLE.md
git add kanban/archives/bug_fixes/BUG_XXX_TITLE.md
git commit -m "docs: Add BUG_XXX fix documentation"
```
