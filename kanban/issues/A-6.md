## 任务目标
为所有页面添加正确的 Schema.org JSON-LD、Meta 标签、robots.txt、sitemap.xml。

---

## 子任务清单
- [ ] 为首页添加 Organization、WebSite、SearchAction Schema
- [ ] 为平台页添加 SoftwareApplication、AggregateRating、BreadcrumbList Schema
- [ ] 为 FAQ 页添加 FAQPage Schema
- [ ] 为 Wiki 页添加 Article/TechArticle Schema
- [ ] 为指南页添加 HowTo Schema
- [ ] 为对比页添加 ItemList Schema
- [ ] 创建 site/robots.txt（Allow /；Sitemap URL）
- [ ] 创建 site/sitemap.xml（包含所有页面）
- [ ] 添加 og: 和 twitter: Meta 标签
- [ ] 验证所有 Schema 通过 Google 结构化数据测试

---

## 验收标准
1. 所有 Schema 通过 Google 结构化数据测试
2. robots.txt 和 sitemap.xml 正确配置
3. Meta 标签完整（og:, twitter:, description, keywords）
4. 页面标题和描述唯一且符合 SEO 规范
