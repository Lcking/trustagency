## 🎙️ VoiceOver 基础快捷键 (macOS)

### 启动/关闭
| 快捷键 | 功能 |
|--------|------|
| Cmd + F5 | 启用/禁用 VoiceOver |
| VO + U | 打开 VoiceOver 工具窗口 |

**注**: VO = Control + Option (默认)

### 导航
| 快捷键 | 功能 |
|--------|------|
| VO + ↑ | 上一个元素 |
| VO + ↓ | 下一个元素 |
| VO + ← | 进入上层容器 |
| VO + → | 进入下层容器 |
| VO + 空格 | 激活当前元素 |
| VO + ` (反引号) | 打开转子菜单 |

### 转子菜单 (VO + `)
允许快速导航到特定元素类型:
- Headings - 按标题导航
- Links - 查看所有链接
- Form Controls - 查看所有表单字段
- Images - 查看所有图像

### 快速导航
| 快捷键 | 功能 |
|--------|------|
| H | 下一个标题 |
| Shift + H | 上一个标题 |
| L | 下一个链接 |
| Shift + L | 上一个链接 |
| B | 下一个按钮 |
| Shift + B | 上一个按钮 |
| F | 下一个表单控件 |
| Shift + F | 上一个表单控件 |
| I | 下一个图像 |
| Shift + I | 上一个图像 |

## 🔧 常见问题排查

### 问题 1: VoiceOver 无法启动

**症状**: Cmd + F5 没有反应

**解决方案**:
1. 进入系统偏好设置 → 无障碍
2. 左侧选择 VoiceOver
3. 勾选 "启用 VoiceOver"
4. 或使用 Cmd + F5

### 问题 2: 元素没有被正确宣读

**症状**: 按钮只说 "按钮"，没有说出文本

**原因**: 缺少可访问的标签

**解决方案**:
```html
<!-- ❌ 不好 - 没有标签 -->
<button><img src="icon.svg"></button>

<!-- ✅ 好 - 有 aria-label -->
<button aria-label="搜索">
  <img src="icon.svg" alt="">
</button>

<!-- ✅ 或用文本 -->
<button>搜索</button>
```

### 问题 3: 转子看不到某些元素

**症状**: H 键找不到某个标题

**原因**: 可能使用了 div 或 span，而不是 h1-h6

**解决方案**:
```html
<!-- ❌ 不好 -->
<div class="heading">标题</div>

<!-- ✅ 好 -->
<h2>标题</h2>
```
