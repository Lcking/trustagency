## 🚀 如何开始使用

### 第一步：克隆项目
```bash
git clone git@github.com:Lcking/trustagency.git
cd trustagency
```

### 第二步：查看文档
```bash
# 快速参考
cat DOCS_ORGANIZED.md

# 详细导航
cat kanban/DOCUMENTS_INDEX.md

# 上传报告
cat GITHUB_UPLOAD_COMPLETE.md
```

### 第三步：启动项目
```bash
# 使用 Docker Compose
docker-compose up -d

# 或运行启动脚本
bash start.sh
```

### 第四步：访问应用
- 前端: http://localhost:3000
- 后端: http://localhost:8000
- API 文档: http://localhost:8000/docs
