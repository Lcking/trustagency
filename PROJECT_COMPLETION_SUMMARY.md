## 🔧 技术实现

### 后端技术栈
- **框架**: FastAPI (Python 3.10)
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: SQLAlchemy
- **认证**: JWT Token
- **API文档**: Swagger/OpenAPI

### 前端技术栈
- **语言**: HTML5 + JavaScript + CSS3
- **编辑器**: Tiptap v2.3.0 (CDN)
- **状态管理**: localStorage
- **认证**: Bearer Token
- **UI框架**: 原生HTML

### 关键特性
- ✅ RESTful API 设计
- ✅ JWT 认证机制
- ✅ 全局错误处理
